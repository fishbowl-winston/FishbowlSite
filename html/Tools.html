TOOLS

Data Mart

The Grants Data Mart is a collection of relevant grants data, distilled from G5. It will allow users to view and sort grants data in an easy to use interface. Users can also connect to the data to run their own analysis.

The Grants Data Mart is supported by documentation including a data dictionary and table maps so users can learn about the structure and types of information.

Entity Risk Review

The Entity Risk Review (ERR) is a pre-risk assessment tool for grant awards. The ERR keeps our Grant making process in compliance with various governance rules and regulations.

Grantee risk is asses in multiple categories to include administration, financial, late audit, state audit, internal controls. In FY15, over 98% of recommended discretionary dollars used an ERR during assessment.
